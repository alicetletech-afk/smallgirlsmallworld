# AliceJens Studio 🐰

สร้างสรรค์พื้นที่ทำงานของเลขาส่วนตัว AI ที่น่ารัก แต่ดูสง่างาม

## ✨ Features

- 📊 Dashboard with real-time stats
- 🧠 Memory management system
- 💬 Customizable AI personality and rules
- 🔔 Smart reminders and calendar integration
- 📝 Interaction logs and timeline
- 🌙 Dark mode support
- 📱 Fully responsive design

## 🎨 Design

- Cute but premium aesthetic
- Soft glassmorphism effects
- Purple / White / Soft Blue palette
- Minimal and modern SaaS style

## 🚀 Getting Started

Simply open `index.html` in your browser or visit the GitHub Pages URL.

### No dependencies required
- Vanilla HTML, CSS, and JavaScript
- Ready to integrate with Google Apps Script

## 📦 Files

- `index.html` - Main application structure
- `style.css` - Complete styling system with dark mode
- `script.js` - Vanilla JavaScript for interactivity
